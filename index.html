ChordPro Reference
Chords
[C]Inline chord — placed directly before the syllable it falls on
[Am] [F#m7]Any chord name: major, minor, 7th, sus, add, dim, aug, etc.
[C/E]Slash chord — bass note after the slash
Song Metadata
{title: My Song}Song title — displayed large at the top
{t: My Song}Shorthand for title
{subtitle: Live Version}Secondary title line — shown below the title in italics
{st: …}Shorthand for subtitle
{artist: Name}Artist or performer name
{composer: Name}Music composer — shown separately if different from artist
{lyricist: Name}Lyrics writer
{key: G}Key signature — shown in header badge and footer chip
{capo: 2}Capo position — shown in meta line and footer chip
{tempo: 120}Tempo in BPM — shown in footer chip
{bpm: 120}Shorthand for tempo
{time: 4/4}Time signature — shown in footer chip
{duration: 3:45}Song duration — shown in meta line
{album: Album Name}Album name — shown below artist
{year: 2024}Release year — shown alongside album
{ccli: 1234567}CCLI song number
{copyright: © 2024 …}Copyright line
Sections
{start_of_verse}Begin a verse — auto-numbered Verse 1, Verse 2…
{start_of_verse: Intro}Begin a verse with a custom label
{end_of_verse}End a verse
{sov} / {eov}Shorthands for start / end of verse
{start_of_chorus}Begin a chorus — auto-numbered after the first
{start_of_chorus: Refrain}Begin a chorus with a custom label
{end_of_chorus}End a chorus
{soc} / {eoc}Shorthands for start / end of chorus
{start_of_bridge}Begin a bridge
{end_of_bridge}End a bridge
{sob} / {eob}Shorthands for start / end of bridge
{start_of_tab}Begin a tab / instrumental section
{end_of_tab}End a tab section
{comment: Label}Standalone labelled section — e.g. {comment: Instrumental}
{c: Label}Shorthand for comment
{chorus}Repeat the last defined chorus inline — shown dimmed
{chorus: x2}Repeat chorus with a custom label
{start_of_tab}Begin a tab section — rendered in monospace font
{end_of_tab}End a tab section
{sot} / {eot}Shorthands for start / end of tab
{start_of_grid}Begin a chord grid section
{end_of_grid}End a chord grid section
{sog} / {eog}Shorthands for start / end of grid
{comment_italic: …}Same as comment
{comment_box: …}Same as comment
{highlight: …}Same as comment
Layout
{new_page}Force a page break — shown as a dashed line in the preview
{np}Shorthand for new_page
{new_song}Mark the start of a new song — resets verse/chorus counters
{ns}Shorthand for new_song
(blank line)Ends the current section and adds vertical spacing
